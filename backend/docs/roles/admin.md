# Admin

Last updated: 2026-03-26

## Overview

Admins have full platform access. They manage the team, positions, candidates, and system settings.

## Team Management

- Invite new admins by email. Invitation expires after 3 days if not accepted.
- Invite new recruiters by email, optionally assigning positions at invitation time.
- View all admins and all recruiters in the system.
- View a recruiter's details and their assigned positions.
- Activate or deactivate any user account (admin or recruiter). Deactivated users cannot log in even with valid SSO credentials.
- Cannot deactivate their own account.
- View all pending user invitations.

## Position Management

- Create positions with a title, description, and one or more assessments. Optionally attach a file and mark the position as external (visible on the public careers page) or internal.
- Edit position details, assessments, and attached files.
- Activate or deactivate positions. Only active external positions are visible to public applicants.
- View all positions in the system, including inactive ones.
- View available assessments from the assessment platform.

## Recruiter Position Access

- Grant or revoke position access for any recruiter.
- When positions are synced, access is added for new positions and revoked for removed ones.
- Recruiters can only see and work with positions they have been granted access to.

## Applicant Review

- View all applicants across all positions, with filters by status, position, or search term.
- View individual applicant details.
- Invite an applicant to assessments — this changes their status from PENDING to INVITED and sends assessment invitations for every assessment linked to the position.
- Reject an applicant, optionally providing a reason. Status changes to REJECTED.
- Edit applicant information and attached files.

## Candidate Oversight

- View all candidates invited to assessments by any recruiter.
- Filter candidates by recruiter.
- View assessment results, scores, and completion status per candidate.

## Invite Candidates

- Invite candidates directly to a position by email and name.
- The system creates one invitation per assessment linked to that position.

## Settings

- Manage global recipient emails. These addresses receive notifications when assessments are completed.
- Add, edit, or remove global recipient emails.

## Ticketing

- Create tickets for assessment scheduling and position readiness tracking.

## Navigation

Admins see the following menu:
- Review Candidates
- Manage Positions
- Manage Team
- Invite Member
- Request Assessment
- Global Recipients

## Access Matrix

| Resource | Access |
|----------|--------|
| Positions | All (active and inactive) |
| Applicants | All, across all positions |
| Candidates | All, from any recruiter |
| Team members | Full management (invite, activate, deactivate) |
| Settings | Full access (global recipients) |
| Tickets | Can create |
